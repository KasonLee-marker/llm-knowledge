# 08 - 其他主要模型调研

## 1. 概览

除 OpenAI、Anthropic、Google、Meta 和 DeepSeek 等头部玩家外，市场上还有多个具有重要价值的模型系列，包括：
- **Mistral AI**（法国开源王者）
- **Microsoft Phi**（小型模型标杆）
- **百度文心 ERNIE**（国产商业模型）
- **腾讯混元**（国产商业模型）
- **ChatGLM / GLM-4**（清华智谱）
- **零一万物 Yi**（双语优化）
- **书生浦语 InternLM**（上海 AI Lab）
- **Cohere Command**（企业级 RAG 专项）
- **xAI Grok**（马斯克旗下）
- **Mistral Codestral**（代码专项）

| Mixtral 8x7B | 56B (13B激活) | 32K | Apache 2.0 | 70.6% | $0.24 | $0.24 |
| Mixtral 8x22B | 141B (39B激活) | 65K | Apache 2.0 | 77.8% | $1.20 | $1.20 |

### 2.2 技术特点

**Sliding Window Attention（滑动窗口注意力）：**
- Mistral 7B 采用 SWA，高效处理长序列
- 在 7B 量级内首个超过 LLaMA 2 13B 的模型

**MoE 架构（Mixtral）：**
- 8 专家激活 2，参数效率高
- Mixtral 8x7B 超越 LLaMA 2 70B 且推理成本更低

**Mistral Small 3 亮点：**
- 24B 参数，**Apache 2.0 开源**，可完全自由商用
- MMLU 81%，接近 GPT-4 级别，本地可运行

### 2.3 适用场景

| 场景 | 推荐模型 | 原因 |
|------|---------|------|
| 免费开源微调 | Mistral 7B v0.3 | Apache 2.0，最成熟 |
| 高性价比开源 | Mistral Small 3 24B | Apache 2.0，强能力 |
| 代码生成 | Codestral 22B | 专项代码优化 |
| 高质量 API | Mistral Large 2 | 价格适中，能力强 |

### 3.2 "教科书质量"数据策略

微软 Phi 的核心创新是**数据质量优先**策略：
- 训练数据经过严格筛选，使用"教科书级别"合成数据
- 同等参数量下，Phi 系列通常超越 LLaMA 等通用预训练模型
- Phi-4 (14B) 在 MMLU 84.8%，超越部分 70B 模型

### 3.3 适用场景

| 场景 | 推荐模型 | 原因 |

| 本地高质量推理 | Phi-4 (14B) | 14B 顶级性能 |
| 企业 Azure 部署 | Phi-3 系列 | 与 Azure AI 深度集成 |
| 高效 MoE | Phi-3.5 MoE | 6.6B 激活，能力强 |
