# OpenAI GPT 系列调研

概览：GPT 系列是目前通用度和生态最丰富的商用模型之一，适合对话、生成、代码与多模态任务（部分版本）。

关键版本：
- GPT-3.5：成本较低，适合常规对话和简单生成任务。
- GPT-4：更强的推理与多任务能力，适合关键业务逻辑。
- GPT-4o / Turbo 系列：在延迟与成本上做了优化，适合实时应用。

集成建议：
- 将高价值/复杂请求路由到 GPT-4，低成本请求使用 GPT-3.5 或本地模型缓存。
- 使用 function-calling + 验证减少 hallucination。
- 注意隐私与数据合规（敏感数据不要直接发给外部 API）。
