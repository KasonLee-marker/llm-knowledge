# 06 - 阿里通义千问（Qwen）系列深度调研

## 1. 系列概览

通义千问（Qwen）是阿里云推出的大语言模型系列，由阿里 DAMO Academy 研发。自 2023 年发布以来快速迭代，2026 年 2 月发布的 Qwen 3.5 系列在多项国际基准上超越同等规模开源模型，是目前综合能力最强的中英双语开源模型之一。

| **Qwen3-30B-A3B** | 30B | **3B** (MoE) | 128K | Apache 2.0 | 超高效 MoE |
| **Qwen3-235B-A22B** | 235B | **22B** (MoE) | 128K | Apache 2.0 | 顶级开源 ⭐ |

**Qwen3 核心创新**：
1. **双模式切换**：同一模型支持"思考模式"（类 o1 推理）和"非思考模式"（快速响应）
   ```
   /think  → 开启推理链（适合数学/代码/逻辑题）
   /no_think → 快速直接响应（适合问答/翻译）
   ```
2. **MoE 效率**：Qwen3-235B-A22B 仅激活 22B 参数，却有 235B 的知识容量
3. **多语言**：支持 119 种语言（远超以往版本）

## 8. 部署方案

### 8.1 阿里云 API（推荐生产）

```java
// Spring AI + 通义千问（OpenAI 兼容接口）
// application.yml
// spring.ai.openai.base-url=https://dashscope.aliyuncs.com/compatible-mode
// spring.ai.openai.api-key=your-dashscope-api-key
// spring.ai.openai.chat.options.model=qwen3.5-plus

@Autowired
private ChatClient chatClient;

public String chat(String message) {
    return chatClient.prompt()
        .user(message)
        .call()
        .content();
}
```

### 8.2 本地部署（Ollama）

```bash
# 下载并运行 Qwen3.5
ollama pull qwen3.5:27b
ollama pull qwen3.5:7b

# 运行
ollama run qwen3.5:27b
```

### 8.3 vLLM 私有化部署

```bash
# Qwen3.5-397B 需要 8×A100 80GB
docker run --gpus all -p 8000:8000 \
  vllm/vllm-openai:qwen3_5 \
  --model Qwen/Qwen3.5-397B-A17B \
  --tensor-parallel-size 8
```

**硬件要求**：
- Qwen3.5-397B-A17B: 8×A100 80GB 或 4×H100 80GB
- Qwen3.5-27B-A3B: 1×A100 40GB
- Qwen3.5-7B: 1×RTX 3090
