# Agent Basics

## Introduction to Agents
Agents are entities that can perceive their environment and take actions autonomously. They include both software agents and robotics.

### Types of Agents
1. **Reactive Agents**: Respond to stimuli in their environment.
2. **Deliberative Agents**: Have a model of the world and can plan actions.
3. **Hybrid Agents**: Combine both reactive and deliberative behaviors.

## Key Concepts
- **Environment**: The context in which an agent operates.
- **Sensors and Actuators**: Mechanisms through which agents receive information and effect changes.

## Applications of Agents
- Robotics
- Automated customer service
- Software bots
